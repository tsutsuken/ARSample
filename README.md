#  ARSample

## 概要

ARWorldMapの機能を使い、AR空間上の状態を保存・復元するアプリです。

## 使用技術

- Xcode 14.0.1
- Swift 5
- UIKit
- RealityKit
- ARKit
- ARWorldMap

## 機能一覧

- AR空間上に箱を表示する
- AR空間の状態を保存する（ARWorldMapを利用）
- AR空間の状態を復元する（ARWorldMapを利用）
